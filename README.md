# SignInPage
> 스파르타 커리어 바로인턴 12기 iOS 앱 개발 직무 과제

## 🗒️ 개요
- storyboard 없는 code base `UIKit`으로 구현했습니다.
- `MVC` 아키텍처를 적용했습니다.
- 라이브러리 없이 구현했습니다.
- 가입한 회원 정보는 `Core Data`, 로그인 정보는 `UserDefaults`를 사용하여 저장 구현했습니다.
- `extension`, `enum`, `delegate`를 적극 활용했습니다.
